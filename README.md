## 个人编程语言学习时的笔记整理，主要用来查漏补缺

其实看的书也不少，也做过一些小工具。但没有及时记录下来，然后几个月也不用，很多就忘了。  

我打算用**提问的方式**去记录一些知识点，这样每次阅读笔记的时候，看着目录，想想这个知识点我能不能回答。  

我会持续整理的，觉得对你有用可以先 star 收藏。

用 markdown 书写，建议使用 [Typora](https://www.typora.io/) 打开阅读。
